SO ÄNDERST DU EIN TEMPLATE

1. HTML-Datei im Browser öffnen (oder im Editor)

2. Oben im :root-Block die Variablen anpassen — z.B. --dj-name: "NEUER NAME"

3. Für Tageszeit-Akzent die Zeile --bbp-accent auf den passenden Hex-Code setzen

4. 3D-Render einfügen: Platzhalter-Div durch <img class="render-image" src="deinRender.png"> ersetzen

5. Screenshot machen oder mit Browser-Plugin als PNG exportieren

MOTIV-REFERENZ

3D-Renders von Arms · Legs · Bodies (OBJ → PNG mit Alpha-Kanal). Stil-Referenz: Eric Prydz "Call on Me" Video — Aerobic-Studio + Körperbetonung, aber niemals Comic, kein Kaleidoskop, kein Glitter. Überleitung via Overlay Studio für Freistellung.
